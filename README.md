# FocusTrack-vscode-extension
A lightweight VS Code extension for FocusTrack that helps capture developer activity inside the editor as part of the wider FocusTrack workflow.
 

## Features
- integrates with FocusTrack workflows
- supports lightweight developer activity capture
- designed to work alongside the FocusTrack desktop app

## Included
- sends high-level local IDE telemetry to the FocusTrack desktop app
- supports workspace activity tracking
- tracks file open, file switch, edit, and save events
- tracks debug session start/end events
- tracks task execution events
- tracks terminal open events
- tracks file rename and delete events
- communicates locally over localhost with token-based authentication
